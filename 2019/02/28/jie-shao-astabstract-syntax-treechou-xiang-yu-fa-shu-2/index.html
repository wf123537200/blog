看《人生果实》，思考人生

    最近看了一个豆瓣网红纪录片《人生果实》，给我带来蛮多震撼的。随后，我买了两位老人的书《每天都是小春日和》，《明天也是小春日和》来读，给我带来了蛮多思考和感悟。

    说实话，这部纪录片是我难得认真看过两次的片子，感觉每次看都有新的东西，虽然片子只是简单的记录了津端家两位老人的生活，但总是感觉有一种情绪从片子里面传达出来，耐人寻味。作为一个理工男，感觉写作已经避免不了用1，2，3，4这样的条目来表达了，这回就省去数字标题，少点逻辑，多点情感。

日子，是两个人的事情

    之前听过很多前辈说：“家里的任何事情，归根结底都是两个人间的事情”。随着年岁增长，我越来越觉得很有道理。片子里面，英子女士以修一先生为中心来过日子，默默的卖掉从娘家带来的东西支持修一先生的帆船爱好，修一先生制作有趣的标识，来提醒妻子各种事情。两个人的日子，既可爱又有趣，虽然平淡如水，却又浓甜如蜜。

    回望自己的日子，很平淡，但不乏有趣，很多事情，真的归根结底是在两个人之间的事情。家么，总有这样那样的事情值得去操心，大到买车买房，小到给孩子报兴趣班，都值得两个人商量，只要两个人的心能靠在一起，互相体谅，就没有过不去的日子。无论贫穷还是富贵，各有各的酸甜苦辣，人家的日子，终究是人家的日子，自己的生活，冷暖自知。

    珍惜和妻子在一起的日子，珍惜孩子长大的时光，珍惜和父母在一起的岁月。牵起值得携手的人，相互理解体谅，度过有趣的“日子”吧。

生活，需要积存时间

    和英子女士的料理一样，和修一先生的日记一样，生活需要长期的累积和坚持。和酒一样，某件事情坚持的久了，自然就有了不一样的味道。

    英子女士日复一日的做她的料理，西餐，日本菜，果酱，等等等等，甚至形成了自己的食谱。我觉得，不仅味道是积累和实践出来的，生活的情趣也需要累积。

    修一先生的日记，一本一本，只记录快乐的事情，一写就是十几年。也是很需要坚持和韧性，待来日回头看看，过往足迹可寻，也是很有趣的。

    记得妈妈把我从小到大的鞋子，我的日记本，我从小到大的书都收集起来，堆在面积不大的家里。原本我还暗暗笑她什么“破烂”都要收起来，现在我和老婆也开始留着儿子的小鞋子了。这可能也是积存时间的生活吧，回望来路，虽然时光匆匆，但如果能看到过往足迹，也是很有意思的事情吧。

相处，不要尝试改变别人

    人啊，下意识里都觉得自己是对的，尝试改变他人，本来就是一个比较难的事情。要改变他人，很容易引起冲突。

    就像英子女士大咧咧的性格一样，严谨的修一先生从没试着改变她，而是改变自己，给妻子带来便利，我想，这需要很多的智慧，也需要很多的爱意吧。

    所以，无论和谁相处，妻子，父母，甚至孩子，都别尝试按照自己的想法去改变他们，如果有爱，为了对方改变自己，可能才是最好的方式。

人生短短，人生长长

    人生说短也短，说长也长，算起来不过几万天，眨眼白驹过隙，人老体衰，也有时候度日如年。所以人生的长度，看起来还是在于精神和感觉上，只要有意思，一世也可以很短，如果没意思，一天也可以很长。

    对英子女士和修一先生来说，恐怕65年的婚姻都有滋有味，如弹指一瞬。重要的可能不是日子的长短，而是和谁过了哪些时光，过得是不是有滋味。如有滋味，人生总是感觉没有过够的吧。

    回想去年的总结里面，我有各种焦虑，焦虑和索求带来的是日子越变越长，越来越没意思。想想，其实有什么可焦虑的呢，过好每一天，让日子有意思才是最重要的把。

    最后，用电影里面的作结吧，‘风吹枯叶落,落叶生肥土,肥土丰香果,孜孜不倦,不紧不慢’，人生果实，自然循环，日子智慧，尽在其中。
