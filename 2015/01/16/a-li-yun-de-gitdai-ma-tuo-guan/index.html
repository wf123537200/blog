阿里云的git代码托管

吐槽啊，阿里云的github简直是慢到家了，各种不能用。 pull pull不下来，push push 不上去。
重试多次可以成功，不过以后不能老是这么处理。

突然灵光一现，将代码转到国内的git源上，coding的git源。 哈哈。这下连上了。 终于代码可以做托管了，以后可以简单的对代码进行编辑了。

而且容易对ghost进行升级，不怕出问题。

同时，记录一下git的使用方式，由于没有指定从哪个分支进行下载，所有的 git pull,git push都会报错为： You asked me to pull without telling me which branch you

需要指定分支下载： git pull origin master
git push origin master
